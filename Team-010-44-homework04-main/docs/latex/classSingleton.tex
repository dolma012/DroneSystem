\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classSingleton}{Singleton}} class defines the {\ttfamily Get\+Instance} method that serves as an alternative to constructor and lets clients access the same instance of this class over and over. 

Note\+: \mbox{\hyperlink{Singleton_8h_source}{Singleton.\+h}} and Singleton.\+cc were derived from the \mbox{\hyperlink{classSingleton}{Singleton}} code example providied by Refactoring.\+Guru. The base code was then modified to fit our extension.
